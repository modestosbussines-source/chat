Guia completo de configurações disponíveis na plataforma OMNI.

## Configurações Gerais

Acesse **Configurações** → **Geral** para ajustar as configurações básicas da sua conta.

### Informações da empresa

| Campo | Descrição |
|-------|-----------|
| **Nome da empresa** | Nome exibido em comunicações e relatórios |
| **Fuso horário** | Define horários de relatórios e agendamentos |
| **Idioma** | Idioma padrão da interface |
| **Logo** | Logo exibida no widget de chat e comunicações |

### Funcionamento

| Configuração | Descrição |
|--------------|-----------|
| **Horário de atendimento** | Define quando o chatbot está ativo |
| **Fora do horário** | Mensagem automática enviada fora do horário |
| **Dias de folga** | Feriados e dias sem atendimento |

## Configurações de Canais

### WhatsApp Business

Acesse **Configurações** → **Canais** → **WhatsApp**

<Aside type="note">
  Nossa equipe configura a conexão inicial durante o onboarding.
</Aside>

**Configurações disponíveis:**

- Número conectado
- Mensagem de boas-vindas
- Horário de envio de mensagens
- Configurações de entrega

### Instagram

Acesse **Configurações** → **Canais** → **Instagram**

**Configurações disponíveis:**
- Conta conectada
- Auto-resposta para stories
- Filtros de mensagem

### Email

Acesse **Configurações** → **Canais** → **Email**

**Configurações disponíveis:**
- Endereço de email
- Assinatura de email
- Regras de encaminhamento

## Chatbot e Automação

### Configurações gerais

Acesse **Chatbot** → **Configurações**

| Configuração | Descrição |
|--------------|-----------|
| **Status** | Ativar/desativar chatbot |
| **Mensagem de boas-vindas** | Primeira mensagem para novos contatos |
| **Mensagem de fallback** | Quando o bot não entende |
| **Horário de funcionamento** | Quando o bot responde |

### Regras de palavras-chave

Crie respostas automáticas baseadas em palavras-chave:

<Steps>

1. Acesse **Chatbot** → **Palavras-chave**
2. Clique em **Nova Regra**
3. Defina a palavra-chave ou frase
4. Configure a resposta (texto, template, ou transferência)
5. Defina a prioridade da regra

</Steps>

### IA Conversacional

Configure respostas com inteligência artificial:

<Steps>

1. Acesse **Chatbot** → **IA**
2. Selecione o provedor (OpenAI, Anthropic, Google)
3. Insira sua chave de API
4. Configure o prompt do assistente
5. Defina gatilhos para ativação

</Steps>

<Aside type="caution">
  O uso de IA tem custo adicional baseado no volume de mensagens processadas. Consulte sua tabela de preços.
</Aside>

## Gestão de Equipe

### Funções e Permissões

Acesse **Configurações** → **Permissões**

**Funções padrão:**

| Função | Descrição |
|--------|-----------|
| **Admin** | Acesso total ao sistema |
| **Supervisor** | Gerencia equipe e visualiza relatórios |
| **Atendente** | Atende conversas e gerencia contatos |

**Permissões disponíveis por recurso:**
- Conversas (ver, responder, transferir, deletar)
- Contatos (ver, criar, editar, deletar)
- Campanhas (ver, criar, enviar)
- Relatórios (ver próprios, ver todos)
- Configurações (acesso total ou limitado)

### Equipes

Organize sua equipe em times especializados:

<Steps>

1. Acesse **Configurações** → **Equipes**
2. Crie equipes por especialidade (Vendas, Suporte, etc.)
3. Atribua atendentes a cada equipe
4. Configure regras de roteamento por equipe

</Steps>

## Notificações

Configure como sua equipe recebe alertas:

| Tipo | Canais |
|------|--------|
| **Nova conversa** | Email, push, Slack |
| **Mensagem não respondida** | Email, push |
| **Transferência recebida** | Push, som |
| **Relatório diário** | Email |

Acesse **Configurações** → **Notificações** para personalizar.

## Integrações

### Webhooks

Configure webhooks para enviar eventos para sistemas externos:

```json
{
  "url": "https://sua-api.com/webhook",
  "events": [
    "conversation.created",
    "conversation.assigned",
    "message.received"
  ]
}
```

## Backup e Exportação

### Exportar dados

Exporte seus dados a qualquer momento:

| Tipo | Formato |
|------|---------|
| Contatos | CSV, JSON |
| Conversas | JSON |
| Relatórios | PDF, CSV |

Acesse **Configurações** → **Dados** → **Exportar**

### Retenção de dados

| Dado | Retenção padrão |
|------|-----------------|
| Conversas | 24 meses |
| Logs de acesso | 12 meses |
| Mensagens | 12 meses |

<Aside type="note">
  Períodos de retenção podem ser ajustados no plano Enterprise.
</Aside>
